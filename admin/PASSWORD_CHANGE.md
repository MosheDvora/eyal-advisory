# מדריך לשינוי סיסמת ממשק הניהול

## הסיסמה הנוכחית
**סיסמה:** `admin123`

## איך לשנות את הסיסמה

### שלב 1: צור Hash חדש לסיסמה
1. לך לאתר: https://emn178.github.io/online-tools/sha256.html
2. הזן את הסיסמה החדשה שלך (לדוגמה: `myNewPassword123`)
3. לחץ על "Hash"
4. העתק את הערך המוצפן (64 תווים)

### שלב 2: עדכן את הסיסמה בקוד
1. פתח את הקובץ: `admin/index.html`
2. מצא את השורה (שורה 209):
   ```javascript
   const ADMIN_PASSWORD_HASH = '240be518fabd2724ddb6f04eeb1da5967448d7e831c08c8fa822809f74c720a9'; // admin123
   ```
3. החלף את הערך בין המרכאות ב-Hash החדש שיצרת
4. עדכן את ההערה (לאחר //) לסיסמה החדשה

### דוגמה:
אם הסיסמה החדשה היא `SecurePass456` ו-Hash שלה הוא `abc123...xyz789`:

```javascript
const ADMIN_PASSWORD_HASH = 'abc123...xyz789'; // SecurePass456
```

### שלב 3: שמור ופרסם
1. שמור את הקובץ `admin/index.html`
2. עשה commit ו-push ל-Git
3. Netlify יעדכן את האתר אוטומטית
